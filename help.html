Desk Planner – User Guide

Overview

The Desk Planner allows you to visually assign desks to teams across the week using an interactive floor plan. You can import teams, allocate desks, and export your plan for reuse or reporting.

---

Main Toolbar

📋 Copy Copies all desk allocations from the selected day to one or more other days.
👥 Import Teams Imports a CSV file containing team IDs and names (e.g. T1, Finance). This builds the team legend automatically.
⬇ Export Exports the current desk plan to CSV, including team names. This file can be re-imported later.
↩ Undo Reverts the last change made to desk assignments.
🗑 Reset Completely resets the application, clearing all data and reloading the page.
🧩 Multi Toggles multi-select mode. When ON, you can select multiple desks without holding Ctrl.
📂 Import Imports a previously exported desk plan CSV. Restores both desk allocations and team names.
➕ Add Team Adds a new team manually. The system automatically assigns the next available team ID (e.g. T1, T4, etc).
🧹 Clear All Clears all desk allocations and removes all teams, without reloading the page.
🧽 Clear Desks Clears desk assignments only. If desks are selected, only those desks are cleared. If none are selected, the entire day is cleared.
📸 Export Image Exports the current floor plan as an image, including team colours and allocation summary.
❓ Help Opens this help guide.
---

Basic Workflow

  • Select a day
  • Add Teams by Import (CSV) or by using the Add team button
  • Click desks on the floor plan either one at a time or many (Multi button or CTRL key) at a time.
  • Once a desk or many desks are selected click on the Team name above under the Teams menu. This will assign the selected desk/s to that team.
  • Click a team in the legend to assign
Tip: Use Multi or hold Ctrl to select multiple desks.
---

Teams

  • Import teams using a CSV file
  • Add new teams using the Add Team button
  • The legend updates automatically
---

Desk Allocation Summary

The Desk Allocation Summary shows:

  • Total desks
  • Assigned vs free desks
  • Desk count per team
---

Compare Mode

Switch to Compare mode to view two days side-by-side. Editing is disabled in this mode.

---

Tips

  • Export regularly to save your work
  • Use consistent team IDs across floors
  • Use Compare mode to review changes
